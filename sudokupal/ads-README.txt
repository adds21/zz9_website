

To view htlm files in visual Code, 

Open VS Code.
Go to the Extensions view by clicking on the square icon in the sidebar or pressing Ctrl+Shift+X (Windows/Linux) 
or Cmd+Shift+X (Mac).
Search for "Live Server" and install it.


Right-click anywhere in the file and select "Open with Live Server".
Alternatively, click on the Go Live button at the bottom-right corner of the VS Code window.
